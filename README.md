## ğŸš€ Firebase Setup for Flutter App (Flutter Selected in Firebase Console)

> âš ï¸ **Important:** This project was created by selecting **Flutter** as the app type in the Firebase Console.

### Step 1: Create Firebase Project

1. Go to **Firebase Console**
2. Click **Add Project**
3. Enter project name
4. Enable Google Analytics (optional)

### Step 2: Add Flutter App to Firebase

1. In Firebase Console, click **Add App**
2. Select **Flutter** (not Android or iOS individually)
3. Firebase shows platform icons (Android, iOS, Web, Windows, etc.)
4. Firebase expects configuration via **FlutterFire CLI**

ğŸ“Œ **No manual download of `google-services.json` or `GoogleService-Info.plist` is required** when using FlutterFire CLI.

* Register app
* Download configuration file:

  * Android â†’ `google-services.json`
  * iOS â†’ `GoogleService-Info.plist`
* Place the file in the correct platform folder

### Step 3: Install Required Dependencies

Add the following to `pubspec.yaml`:

```yaml
dependencies:
  flutter:
    sdk: flutter
  firebase_core: ^3.0.0
  firebase_auth: ^5.0.0
```

## ğŸ”¥ Firestore Real-Time Data Integration

Firestore syncs data **automatically and in real time** across all connected devices.

### Firestore Collection Example:

```dart
final CollectionReference tasks =
    FirebaseFirestore.instance.collection('tasks');

Future<void> addTask(String title) {
  return tasks.add({
    'title': title,
    'createdAt': Timestamp.now(),
  });
}

Stream<QuerySnapshot> getTasks() {
  return tasks.orderBy('createdAt', descending: true).snapshots();
}
```

### UI with Real-Time Updates:

```dart
StreamBuilder(
  stream: getTasks(),
  builder: (context, snapshot) {
    if (!snapshot.hasData) return CircularProgressIndicator();
    final docs = snapshot.data!.docs;
    return ListView(
      children: docs.map((doc) => Text(doc['title'])).toList(),
    );
  },
);
```

ğŸ“Œ When one user adds data, all users see it instantly without refresh.
